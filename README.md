# Fashion Aura

A responsive multi-page fashion store website built using HTML5 and CSS3. Fashion Aura showcases modern fashion collections with a clean, user-friendly, and visually appealing interface.

## Live Website

https://jaguu-tech.github.io/DecodeLabs_Static_WebPage_Fashion-Aura/

## GitHub Repository

https://github.com/Jaguu-tech/DecodeLabs_Static_WebPage_Fashion-Aura.git


## Project Overview

Fashion Aura is a fashion store website designed to provide users with an elegant browsing experience. The website features multiple collection pages, responsive layouts, and a modern fashion-inspired design.

This project was developed as part of the **Frontend Development Internship at DecodeLabs**.


## Features

- Responsive Design
- Modern Fashion Store Interface
- Multi-Page Website
- Hero Banner Section
- Women's Collection Page
- Men's Collection Page
- Accessories Collection Page
- Footwear Collection Page
- Product Cards with Images and Pricing
- Clean Navigation Menu
- Footer Section
- External CSS Styling
- Mobile-Friendly Layout
